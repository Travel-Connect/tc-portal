# /verify - ビルド検証コマンド

以下を順に実行し、結果を要約してください：

1. **Lint チェック**
   ```bash
   npm run lint
   ```

2. **型チェック**
   ```bash
   npx tsc --noEmit
   ```

3. **ビルド**
   ```bash
   npm run build
   ```

## 出力形式

```
## 検証結果

| チェック | 結果 |
|----------|------|
| Lint | ✅ / ❌ (エラー数) |
| TypeCheck | ✅ / ❌ (エラー数) |
| Build | ✅ / ❌ |

### エラー詳細（あれば）
- [ファイル:行] エラー内容
```
